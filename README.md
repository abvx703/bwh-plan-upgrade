## 差价怎么算：不是重新付全款

这是很多人搞不清楚的地方。搬瓦工升级**只补差价**，不需要重新付新套餐的完整费用。

计算公式是：

> **差价 = （新套餐价格 - 旧套餐价格）× （剩余天数 ÷ 付款周期天数）× 优惠码折扣**

举个例子。年付 $49.99 的 KVM 套餐，用了 60 天后想升到 $169.99/年 的 CN2 GIA-E，假设当时买的时候用了 6% 的优惠码：

> 差价 ≈ (169.99 - 49.99) × (305 ÷ 365) × 0.94 ≈ **$94.3**

不是 $169.99，只是剩余周期里的差值部分。

还有一个小技巧：升级时如果找到了折扣力度更大的优惠码，可以在支付页面的优惠码栏替换一下。续费不支持换码，但升级这步可以。

## 升级后做这几件事

套餐升级完成、账单付掉之后，还有几步建议跑一下：

1. **登录 KiwiVM 控制面板，手动重启一次**。系统可能会自动重启，但手动 Reboot 一次能确保新分配的内存和 CPU 配置完全加载到位。

2. **用命令行核对资源是否更新**。SSH 登进去跑一下 `free -m`（看内存）和 `df -h`（看硬盘），确认数值已经变成新套餐的配置。如果跑了之后数字还是旧的，就是没重启干净，再 Reboot 一次。

3. **跑一次 KiwiVM 的快照**。升级之后这台机器的状态是最"干净"的，创建一个快照当作还原点，以后哪天搞坏了还能回滚。

4. **如果换了机房（线路），测一下连通性**。从国内 ping 一下新 IP，确认延迟正常，三网路由没问题。

## 几个容易踩的坑

**坑一：升级跨系列后 IP 可能变**

同系列升级（比如 KVM 低配升 KVM 高配）IP 不变。但如果从普通 KVM 跨系列升到 CN2 GIA 系列，机房本身换了，IP 大概率也会换。升之前查一下目标套餐对应的机房位置，再决定要不要升。

**坑二：降级不是想降就能降**

目前搬瓦工自助降级的支持比较有限，部分套餐需要联系客服处理，不是所有方案都能降。如果你买的是性价比高的限量版套餐，升了之后真的很难降回来。

**坑三：流量用完不等于要升套餐**

月流量跑完了，等下个月自动重置就行，不需要升配置。流量的问题是临时性的，配置的问题才需要升套餐来解决。

**坑四：买了限量版不要轻易升**

搬瓦工不定期出限量版套餐，价格比常规套餐便宜不少（比如 $39/年 就能拿到 CN2 GIA 线路）。这类套餐一旦升级，就进入了常规套餐体系，以后续费按常规价格走。

## FAQ

**Q：搬瓦工升级方案需要重装系统吗？**

不需要。升级全程数据完整保留，系统不会重装，环境和配置都在，只是配置资源（内存、流量、硬盘）会扩容，可能会有一次短暂重启。

**Q：升级时可以顺便换优惠码吗？**

可以。在支付差价的界面有优惠码输入框，如果手头有比当初买时更优惠的码，可以在这步换掉。续费的时候不支持换码，只有升级这步能改。

**Q：从 KVM 普通套餐能直接升到香港 CN2 GIA 吗？**

一般可以，系统会列出可升级的目标套餐。但最终以升级页面实际显示的选项为准，系统不会显示的就是不支持跨升。

**Q：升级后的差价怎么付？支付宝可以吗？**

系统自动生成账单，支持支付宝付款，和买套餐时的付款流程一样，操作上没区别。

**Q：我手上是一个限量版套餐，升了之后还能降回去吗？**

大概率不能。搬瓦工降级本身就不完全支持自助操作，限量版套餐升级后更没有"降回原价"这种选项。所以这个决定做之前想好。
